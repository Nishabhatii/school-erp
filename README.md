# 🏫 School ERP System

A simple web-based School ERP system to manage student,teacher, and admin activities.
 
## 🚀 Features
- 🧑‍🎓 Student Management
- 👩‍🏫 Teacher Login
- 📅 Timetable and Attendance
- 📊 Reports and Dashboard

## 🛠 Tech Stack
- React JS (Frontend)
- CSS (Styling)
- Git & GitHub (Version Control)
